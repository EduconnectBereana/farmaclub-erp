🔑 Passo 1 — Chave do Supabase

No Supabase, acesse: Project Settings → API → Project API keys
Copie a chave chamada "anon public" e cole abaixo.

Onde encontrar no Supabase:
1. Acesse supabase.com → seu projeto
2. Menu esquerdo: clique em ⚙️ Settings
3. Clique em API
4. Em "Project API keys", copie a anon public (NÃO é a service_role)
5. Cole no campo acima e clique em "Testar conexão"

🏥 Passo 2 — Farmácias parceiras

Cadastre cada farmácia da rede. As coordenadas GPS são usadas para calcular o frete e ordenar os motoboys.

💡 Para pegar as coordenadas GPS: abra maps.google.com, pesquise o endereço da farmácia, clique com o botão direito no mapa e copie os números que aparecem (ex: -21.2062, -41.8878)

👤 Passo 3 — Usuários e acessos

Crie um login (e-mail + senha) para cada farmácia parceira. Também crie o seu acesso como gestor da rede.

📧 O e-mail precisa ser válido — o Supabase envia um link de confirmação. Após confirmar o e-mail, o usuário pode logar no ERP. Para pular a confirmação em testes, ative "Disable email confirmation" em Authentication → Settings no Supabase.
